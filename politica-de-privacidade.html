Política de Privacidade

Esta política explica como a Horizon Digital recolhe, utiliza e protege as informações partilhadas pelos visitantes e clientes deste site.

1. Dados que recolhemos

Recolhemos apenas os dados que nos fornece diretamente ao entrar em contacto, como nome, número de WhatsApp e detalhes do serviço pretendido.

2. Como usamos os dados

Os dados são usados exclusivamente para processar pedidos, prestar os serviços contratados e comunicar contigo sobre o andamento do teu pedido.

3. Partilha de dados

Não vendemos nem partilhamos os teus dados com terceiros, exceto quando necessário para a prestação do serviço (por exemplo, plataformas de cartões ou streaming).

4. Segurança

Aplicamos boas práticas de segurança, incluindo ligação HTTPS, para proteger a informação transmitida através deste site.

5. Contacto

Para questões sobre esta política, contacta-nos em horizondigitao@gmail.com.
